A Snake Game

In my first Semester, my coursemate and I created a Snake Game. The Snake Game is a classic arcade-style game where the player controls a snake on a grid. Using the same concept, the objective of the game is to eat food pellets, which causes the snake to grow longer. The player must navigate the snake around the grid, avoiding collisions with the walls or its own body. As the snake grows, it becomes more challenging to maneuver without colliding. The game ends when the snake collides with a wall or itself by showing “SORRY, YOU DIE! SCORE: “ Some of the concepts I utilized when developing this game include, classes and objects, such as the Button class for the start screen and the Snake class for managing the snake's position and movement. I used the loop that continuously updates the game state, handles user input, and renders the graphics. The game generates random positions for the food pellets, ensuring that they appear in different locations each time. This randomness adds variety and challenge to the gameplay.
